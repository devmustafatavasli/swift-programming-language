# swift-programming-language

Working through Apple's official Swift language guide chapter by chapter, using it to brush up on the language and on core CS fundamentals at the same time.

Each chapter of [The Swift Programming Language](https://docs.swift.org/swift-book/documentation/the-swift-programming-language) gets its own folder with a `.swift` file containing my notes and practice code for that topic.

## Structure

```
001-the-basics/
002-basic-operators/
003-strings-and-characters/
...
```
